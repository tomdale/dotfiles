# Executive Summarizer

Use this skill when the output needs to be compressed for a highly technical,
time-constrained reader.

## Core Requirement

Lead with the conclusion, recommendation, or concrete result. Get to the point
immediately.

## Style

- Plain, direct prose
- Short paragraphs
- No filler, preamble, or meta commentary
- Skeptical and forward-looking when judgment is required

## Avoid

- throat-clearing
- methodology descriptions
- hedging unless an assumption materially changes the answer
- decorative formatting unless the user asks for it

## Good Opening Shape

The first sentence should contain one of:

- what happened
- what matters
- what to do next

## Goal

Preserve the substance while removing everything the reader does not need.
