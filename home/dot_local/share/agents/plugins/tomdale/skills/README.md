# Skills

This plugin currently bundles:

- `abstraction-boundaries`
- `benchmarking`
- `cruft`
- `documentation-comments`
- `exec-summarizer`
- `fix-pr-checks`
- `interactive-shell`
- `iterm2-python-api`
- `minify-prompt`
- `narrative`
- `refactoring`
- `ship-pr`
- `tasks`
- `tui-development`
- `write-task`

Add future personal skills as one directory per skill:

```text
skills/
  release-checklist/
    SKILL.md
```

Keep shared helpers in `../scripts/` instead of duplicating them in each skill
directory.
